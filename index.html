JobScout

A full-stack job-hunting application built with JavaScript, React (Vite), Node.js, Express, and MongoDB Atlas. Features include user authentication, real-time updates, and a responsive design. The GPT-3.5-Turbo API is used for generating personalized cover letters based on job descriptions.

JobScout Mobile

A mobile version of JobScout, developed with React Native, providing a seamless user experience. It uses the same backend as the original JobScout web app, ensuring consistent data synchronization and core functionalities across both platforms.

CarBidder

For Car Dealerships that focus solely on selling vehicles without engaging in direct car trade-ins or exchanges. This web application was born to solve the problem for you. Manage Form Submissions all in the single Admin Panel, fix, update and send the forms to your Partners for receiving an offer for the Customer's Vehicle.

Los Santos Custom

A three-phase Java simulation using JavaFX and Engine class to model and simulate customer interactions at a car dealership. Simulation includes 4 service points such as customer service, finance, test drives and sales. Dealership owners can simulate the need for employees at different service points.

View Demo GitHub Repo

Heart Rate Detector (IoT)

This IoT heart rate sensor, built with Raspberry Pi Pico and MicroPython, monitors heart rate and calculates HRV on-board. It wirelessly transmits data via MQTT and integrates with Kubios for advanced analytics, offering real-time monitoring and analysis.
