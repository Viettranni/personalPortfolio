JobScout

A full-stack job-hunting application built with JavaScript, React (Vite), Node.js, Express, and MongoDB Atlas. Features include user authentication, real-time updates, and a responsive design. The GPT-3.5-Turbo API is used for generating personalized cover letters based on job descriptions.

A mobile version of JobScout, developed with React Native, providing a seamless user experience. It uses the same backend as the original JobScout web app, ensuring consistent data synchronization and core functionalities across both platforms.
